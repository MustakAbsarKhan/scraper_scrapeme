# Scrapeme Web Scraper

This is a Python web scraping project that extracts product titles and prices from the Scrapeme website.

## Features
- Scrapes multiple pages automatically
- Extracts product title and price
- Uses requests and BeautifulSoup
- Handles missing data safely

## Technologies Used
- Python
- requests
- BeautifulSoup

## How to Run

1. Install dependencies:
   pip install requests beautifulsoup4 lxml

2. Run the script:
   python scraper.py
